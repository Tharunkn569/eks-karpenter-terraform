# Orders Microservice

## Overview

This folder contains the Helm chart for the Orders Microservice.

## Files

Chart.yaml
values-dev.yaml
values-prod.yaml
templates/

## Purpose

The Orders Microservice is used to demonstrate persistent storage on Amazon EKS using Amazon EBS CSI Driver.
This microservice includes:

StorageClass
PersistentVolumeClaim
EBS-backed volume mount
Environment-specific values files
