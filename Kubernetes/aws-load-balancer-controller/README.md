# AWS Load Balancer Controller

## Overview

This folder contains environment-specific configuration for AWS Load Balancer Controller.

## Files

```text
values-dev.yaml
values-prod.yaml
```

## Purpose

AWS Load Balancer Controller enables Kubernetes Ingress resources to integrate with AWS Application Load Balancers (ALB).

The Payments Microservice uses an Ingress resource that can be managed through AWS Load Balancer Controller.
